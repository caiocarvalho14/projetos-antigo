Nós Somos os Alligators

O projeto AlligatorPlay foi orgulhosamente desenvolvido pelo 3º Ano B do Ensino Médio de Informática, uma iniciativa que une tecnologia e o espírito esportivo do JIS.

Do Código à Competição

Este site é um projeto desenvolvido pela nossa sala, onde aplicamos o conhecimento em desenvolvimento web, banco de dados e gestão de projetos aprendidos em sala de aula para criar uma solução real e útil para nossa comunidade escolar.

O AlligatorPlay transforma a teoria em prática, mostrando como a tecnologia pode ser uma poderosa ferramenta para engajar, informar e celebrar os momentos incríveis dos Jogos Internos do SESI (JIS).
