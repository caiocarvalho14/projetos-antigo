Sobre Mim

Olá, sou o

Um desenvolvedor full-stack em formação pelo curso Técnico em Informática para Internet do SENAI. Minha paixão é usar a tecnologia para resolver problemas e construir soluções web completas, o que me motiva a estar sempre aprendendo. Atualmente, dedico meu foco ao EstágioS, um projeto de larga escala para o gerenciamento de estágios, onde tenho a oportunidade de aplicar na prática meus conhecimentos em front-end, back-end e banco de dados.

Como desenvolvedor, acredito que a qualidade de uma aplicação nasce de uma base sólida e bem planejada. Por isso, sou um grande defensor das melhores práticas de desenvolvimento, utilizando modelagem UML para garantir uma arquitetura robusta e priorizando sempre um código limpo, organizado e escalável. Meu compromisso é estar sempre atualizado com as tecnologias mais modernas, buscando entregar projetos que não sejam apenas funcionais, mas também eficientes e fáceis de manter.

Landing Page Comercial

Landing page criada com Bootstrap, com design responsivo e seções estratégicas para apresentar produtos ou serviços. A interação com o cliente é facilitada por botões que redirecionam diretamente para o WhatsApp do vendedor, permitindo atendimento rápido e personalizado. O projeto combina estética moderna com funcionalidade prática, garantindo uma experiência fluida em diferentes dispositivos.

Ecommerce de doceria

E-commerce funcional para uma doceria, desenvolvido com Bootstrap CSS para garantir um layout moderno e responsivo. Conta com catálogo de produtos, carrinho de compras e cálculo automático do total. A finalização é feita por meio de integração com o WhatsApp, onde uma mensagem automatizada é gerada via JavaScript e enviada diretamente ao responsável, garantindo praticidade e agilidade no atendimento.

Sistema de gestão de estágios

EstagioS é um sistema de gerenciamento de estágios voltado para a escala de jovens aprendizes, com diversas funcionalidades para a gestão de alunos, empresas e processos administrativos. Foi desenvolvido durante a formação no curso técnico de Desenvolvimento Web do SENAI, utilizando Django no backend, PostgreSQL como banco de dados e Tailwind CSS para a interface, garantindo robustez, organização e uma experiência moderna e responsiva.
